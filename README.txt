Инструкция к использованию инструмента:
1. Для открытия инструмента необходимо перейти в верхнем меню в пункт Tools -> Save Points
2. Далее, нажать на кнопку Select/Serialization
3. В появившемся окне выбрать путь, куда сохранить JSON-файл
4. После использования, инструмент также выделит все записанные в файл "точки", существующие на сцене 

Что такое "точка":
Технически - это объект с подключённым скриптом "Point". Он имеет два дополнительных float-поля, которые можно задавать на сцене
